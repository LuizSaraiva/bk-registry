<h1 align="center">
  Bank Project
</h1>

<br>

Project created to practice knowledge of JAVA, Spring Boot, etc.

These modules are part of a simulation of a banking ecosystem.

---

### Modules:
- Account Registry
- History Registry

## Tecnologias / Libraries

- Spring Boot
- Java 17
- Mysql 8
- RabbitMQ
- MapStruct
- Liquibase
- Lombok
